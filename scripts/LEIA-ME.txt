================================================
  IMPRESSORA DE PEDIDOS v3.0 - GUIA DE INSTALACAO
================================================

REQUISITOS:
- Windows 10 ou 11
- Impressora térmica instalada no Windows
- Conexão com a internet

INSTALACAO:

1. Extraia todos os arquivos para uma pasta
   Exemplo: C:\ImpressoraPedidos

2. Baixe o arquivo 'config.ini' do sistema
   - Acesse o sistema web
   - Vá em "Impressoras" no menu lateral
   - Clique em "Baixar config.ini"
   - Salve na mesma pasta do programa

3. Execute o programa
   - Dê duplo clique em "ImpressoraPedidos.exe"
   - A interface gráfica aparecerá
   - O programa ficará minimizado na bandeja do sistema

INTERFACE DO PROGRAMA:

- Indicador verde: Conectado e funcionando
- Indicador vermelho: Problema de conexão
- Log de atividades: Mostra os pedidos impressos
- Menu Opções > Testar Impressão: Imprime página de teste
- Menu Opções > Abrir config.ini: Edita configurações

CONFIGURACAO MANUAL (opcional):

Se precisar editar o config.ini manualmente:

[GERAL]
SUPABASE_URL = sua_url_aqui
SUPABASE_KEY = sua_key_aqui

[RESTAURANTE]
ID = seu_restaurant_id
IMPRESSORA =  (deixe vazio para usar a padrão)

[SISTEMA]
INTERVALO = 5    (segundos entre cada verificação)
LARGURA_PAPEL = 48  (48 para 58mm, 42 para 80mm)

INICIAR COM O WINDOWS:

1. Pressione Win + R
2. Digite: shell:startup
3. Crie um atalho do ImpressoraPedidos.exe nessa pasta

PROBLEMAS COMUNS:

"Windows SmartScreen bloqueou"
  → Clique em "Mais informações" e depois "Executar assim mesmo"

"Nenhuma impressora encontrada"
  → Verifique se a impressora está instalada no Windows

"Erro de conexão"
  → Verifique sua internet e o arquivo config.ini

"Indicador vermelho"
  → O programa não consegue conectar ao servidor
  → Verifique sua internet e o arquivo config.ini

SUPORTE:

Em caso de problemas, entre em contato com o suporte técnico
ou regenere o arquivo config.ini no sistema web.
